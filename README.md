# Smart Trip Companion – Full-Stack Prototype

Smart Trip Companion is a full-stack web application prototype designed to
explore how a digital assistant could enhance the tourist experience during
a trip.

The project focuses on end-to-end product development: frontend UI,
backend APIs, and cloud deployment.

## What this project demonstrates
- Full-stack application architecture
- Frontend–backend integration
- API-driven design
- Product-oriented thinking for tourism use cases

## Tech Stack
- Frontend: React / Next.js
- Backend: Node.js, Express
- Database: MongoDB
- Deployment: Azure App Service
